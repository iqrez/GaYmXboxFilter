# Experimental Tools

This folder contains one-off probes and older verification helpers that were useful during stack discovery but are not part of the curated MVP toolchain.

Examples:

- raw HID layout probes
- early XInput-only verifiers
- temporary enumeration helpers

The supported tools remain in [GaYmTestFeeder](../../GaYmTestFeeder) and are built through [scripts/build-tools.ps1](../../scripts/build-tools.ps1).
